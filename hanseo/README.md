# How To Use

1. venv를 활성화합니다. (https://www.notion.so/How-to-use-RunPod-2463e8485e5980638b2eff260368b974 참고)
2. requirements.txt 설치합니다.
3. model_download.ipynb를 실행해 LLM을 설치합니다.
4. 추론을 위해 run.py를 실행합니다.
